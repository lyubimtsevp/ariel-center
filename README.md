# Сайт Центра коррекции речи и поведения «Ариель»

Современный сайт для медицинского и образовательного центра, разработанный на Next.js 14.

## 🚀 Технологии

- **Next.js 16** - React-фреймворк с SSR и SSG
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **Zustand** - управление состоянием
- **Lucide React** - иконки

## 📁 Структура проекта

```
website/
├── src/
│   ├── app/                 # Страницы (App Router)
│   │   ├── about/          # О центре
│   │   ├── cart/           # Корзина
│   │   ├── contacts/       # Контакты
│   │   ├── faq/            # Частые вопросы
│   │   ├── prices/         # Цены
│   │   ├── services/       # Услуги
│   │   └── specialists/    # Специалисты
│   ├── components/         # React-компоненты
│   │   ├── layout/         # Header, Footer
│   │   ├── sections/       # Секции страниц
│   │   └── ui/             # UI-компоненты
│   ├── data/               # JSON данные (контент)
│   ├── lib/                # Утилиты
│   ├── store/              # Zustand stores
│   └── types/              # TypeScript типы
└── content/                # Исходные данные (вне website)
```

## 🛠 Запуск

```bash
# Установка зависимостей
npm install

# Разработка
npm run dev

# Сборка
npm run build

# Продакшн
npm start
```

## 📦 Функционал

### Реализовано:
- ✅ Главная страница с Hero, услугами, специалистами, FAQ
- ✅ Страница услуг с каталогом
- ✅ Страница специалистов
- ✅ Прайс-лист с возможностью добавления в корзину
- ✅ FAQ с поиском и аккордеоном
- ✅ Страница контактов с формой
- ✅ О центре с лицензиями и реквизитами
- ✅ Корзина с управлением количеством
- ✅ Чат-бот с кнопками и сценариями
- ✅ Адаптивный дизайн
- ✅ SEO оптимизация

### Для полноценного запуска требуется:
- 🔲 Интеграция ЮKassa для оплаты
- 🔲 Настройка отправки форм (email/Telegram)
- 🔲 Загрузка реальных фотографий специалистов
- 🔲 Настройка Yandex.Maps API
- 🔲 Подключение CMS для редактирования контента

## 📝 Контент

Весь контент сайта хранится в JSON-файлах в папке `src/data/`:
- `company.json` - информация о компании, лицензии, реквизиты
- `contacts.json` - контакты, адреса, режим работы
- `specialists.json` - список специалистов
- `services.json` - услуги и направления
- `prices.json` - прайс-лист
- `faq.json` - часто задаваемые вопросы
